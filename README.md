# Yellow Color Detection using Webcam and Python

This is a simple Python program that detects yellow-colored objects using a web cameras.

## Libraries

- Python 3.x
- OpenCV (`opencv-python`)
- NumPy (`numpy`)
- Pillow (`Pillow`)
